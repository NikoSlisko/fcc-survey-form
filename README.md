# FCC Survey Form - Gaming Setup & Habits

Responsive Web Design Certification Project (freeCodeCamp)

**Live Demo:** https://nikoslisko.github.io/fcc-survey-form/

**Beschreibung**  
Eine responsive Umfrage zum Thema Gaming-Setup und Spielgewohnheiten (2026).  
Erfüllt alle Anforderungen der freeCodeCamp Responsive Web Design Certification:  
- Text-, Email-, Number-, Select-, Radio-, Checkbox-, Textarea-Elemente  
- Korrekte Label-Verknüpfung und Accessibility  
- Mobile-First-Design mit Flexbox/Grid

**Technologien**  
- HTML5  



**Hinweis**  
Dieses Projekt ist Teil der freeCodeCamp Responsive Web Design Certification und dient als Portfolio-Beispiel für Bewerbungen im Bereich Web-Entwicklung.
